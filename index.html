Trentoniana


Description

Trentoniana is a museum in the Trenton Free Public Library. As a project for the Design Perspectives Course within the Interactive Multimedia Department we were tasked with creating an interactive kiosk to showcase the artifacts and encourage visitors to interact with them. Groups were assigned to gather documents, videos and maps. My group was tasked with creating the actual application.


Process

  • My group interviewed the staff of the Trenton Free Public Library to gather requirements for the program
  • We divided up the work exploring different map systems to use for the kiosk
  • The other two members worked on implementing the ArcGIS system while I worked with Google Maps
  • Provided a style guide from the design standards group in the class, I created an interactive visual prototype using Sketch
  • Using this prototype we gathered feedback from the staff and used it when creating the actual site

Outcome

I created a higher fidelity prototype using HTML, CSS, JS, and the Google Maps API. The source code can be viewed here. See screenshots of the prototype below.

Handimations


Description

Handimations is a drawing application that utilizes the LeapMotion hand sensor to allow users to draw their own flipbook gifs. No sticky-notes or art supplies needed! This project was developed in Fall 2018 as a final project for CSC 355: Human Computer Interaction at The College of New Jersey.


Process

  • I worked on a team of four so the work was divided up accordingly
  • We familiarized ourselves with the LeapMotion Javascript API
  • I mainly focused on the User Interface and fine tuning the interaction gestures for drawing/erasing on the canvas
  • As the final project evaluation we created a survey and had about 20 people evaluate our application
  • The results confirmed that a majority of the users had fun while using the application and only 2 out of 20 were bothered by the fatigue from drawing

Hyperfocus


Description

Hyperfocus is a web based application that will assist students with learning disabilities centering around Executive Function Disorder (EFD). Executive functions include the cognitive and mental capabilities that allow people to be goal-oriented, focused and organized. Executive function is highly connected to Attention Deficit Disorder (ADD or ADHD). People who have this disorder are more motivated by short-term goals and activities. Approaching this issue from the point of view of a student learning and studying takes a lot more effort for people with EFD and ADD. The web application will act as study guide generator that will allow them to quickly input the information and output a fully formatted and organized study guide.


Process

  • Planned out the base functionality
  • Designed mockups for the user interface
  • Started Learning Ruby on Rails and hosting on a virtual machine
  • Began implementing the base functionality with the start page and the dynamic form
  • Redesigned the user interface to be more minimal and less distracting
  • Finalized prototype by adding pdf export functionality

Iowa State University Undergraduate Research Experience


Description

A web application to teach complex indoor climate control concepts to underserved populations whose homes are not prepared for extreme weather.


Process

  • Since this was an ongoing project, we reviewed past surveys in order to understand the target population and their current knowledge of indoor climate control
  • Conducted a Literature Review to figure out the best way to teach the concepts
  • Created User Stories to gain insight
  • Created paper prototypes to figure out the flow of interaction
  • Using ReactJS we created a functional web application available in two languages
  • Conducted a pilot study through the use of surveys and walked the volunteers through scenarios
  • Created a poster and paper that we submitted for publication
